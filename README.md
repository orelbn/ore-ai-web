# Ore AI

> Current phase: Design and Architecture

A playground for experimenting with AI agent workflows. The agent is built around personal interests, has tools for real hobbies, and is meant to have a distinct personality. Most of the code is written by AI agents; the product direction, prompting, and review are not.

If you want to fork this as a starting point for your own agent, go for it.

## What Makes It a Fun Learning Experience

The agent-first workflow is the experiment. The goal is to get better at agentic coding by actually doing it.
